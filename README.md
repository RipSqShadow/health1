# MaatriTrack - Maternal Healthcare Tracking for Rural India

A comprehensive maternal healthcare web application aligned with UN SDG 3 (Good Health & Well-being).

## Features

- **13 Complete Pages**: Landing, Stats, Dashboard, Login, Registration, Profile, Visits, Immunization, High-Risk, Reports, Education, Emergency, About, Contact
- **23 Indian Languages**: All scheduled languages of India with proper font rendering
- **3D Animations**: Three.js powered 3D scenes throughout the website
- **Problem-Solving Dashboard**: Action-oriented dashboard for healthcare workers
- **Wellness Tracker**: Complete A-Z maternal health tracking (nutrition, water, sleep, mood, exercise, supplements, medication, weight, symptoms, labs, mental health)
- **Emergency Helpline**: One-tap family emergency calls and public health support (108, 102, 104, 181)
- **Auto Risk Detection**: BP, hemoglobin, age-based high-risk pregnancy flagging
- **Mobile-First Design**: Optimized for low-end Android phones and 2G/3G networks

## Tech Stack

- React 18 + TypeScript + Vite
- Tailwind CSS with custom MaatriTrack design system
- Three.js / React Three Fiber for 3D animations
- Framer Motion for UI animations
- Recharts for analytics dashboards
- React Router for navigation

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:5173
